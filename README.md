# RAG Deadlock Detection

This repository contains a Python Jupyter Notebook to detect deadlocks using Resource Allocation Graphs (RAG). The notebook uses `networkx` and `matplotlib` for graph creation and visualization.

## How to Run
1. Install the required packages:
   pip install networkx matplotlib
2. Open the notebook `RAG_Analysis.ipynb` in Jupyter Notebook.
3. Run the cells in order.
4. The last cell lets you choose:
   - 1 → Single interactive case
   - 2 → Multi-case predefined scenarios
